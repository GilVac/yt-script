# yt-script
script for download videos youtube


English

Installation and Usage Instructions:

    Download the script to your preferred directory in the terminal.

    Make it executable by running the command:
    sudo chmod +x install_yt_downloader.sh
    (Using sudo ensures you have the necessary permissions.)

    Run the script to start using it:
    ./install_yt_downloader.sh

    To make it easier to use, you can set up a system-wide command alias. For example, if you want to use yt as the command, you can do one of the following:

        Create a symbolic link in /usr/local/bin/ (requires sudo):
        sudo ln -s /full/path/to/install_yt_downloader.sh /usr/local/bin/yt

        Or, add an alias to your shell configuration file (e.g., ~/.bashrc or ~/.zshrc):
        alias yt='/full/path/to/install_yt_downloader.sh'

    After setting up the alias, you can simply use yt followed by any subcommand. For example, to see the help menu:
    yt help

العربية

تعليمات التثبيت والاستخدام:

    قم بتنزيل السكربت إلى المجلد المطلوب في التيرمينال.

    اجعله قابلاً للتنفيذ باستخدام الأمر:
    sudo chmod +x install_yt_downloader.sh
    (استخدام sudo يضمن حصولك على الصلاحيات اللازمة.)

    شغّل السكربت للبدء باستخدامه:
    ./install_yt_downloader.sh

    لتسهيل الاستخدام، يمكنك إنشاء أمر مختصر يعمل على مستوى النظام. على سبيل المثال، إذا أردت استخدام yt كأمر رئيسي، يمكنك فعل أحد الأمرين:

        إنشاء رابط رمزي في /usr/local/bin/ (يتطلب sudo):
        sudo ln -s /المسار/الكامل/لـ/install_yt_downloader.sh /usr/local/bin/yt

        أو، إضافة اسم مستعار (alias) إلى ملف إعدادات الشل الخاص بك (مثل ~/.bashrc أو ~/.zshrc):
        alias yt='/المسار/الكامل/لـ/install_yt_downloader.sh'

    بعد إعداد الأمر المختصر، يمكنك ببساطة استخدام yt متبوعًا بالأمر الفرعي. على سبيل المثال، لرؤية قائمة المساعدة:
    yt help

فارسی

دستورالعمل نصب و استفاده:

    اسکریپت را دانلود کنید و در پوشه مورد نظر در ترمینال قرار دهید.

    با اجرای دستور زیر آن را قابل اجرا کنید:
    sudo chmod +x install_yt_downloader.sh
    (استفاده از sudo دسترسی‌های لازم را تضمین می‌کند.)

    برای شروع استفاده، اسکریپت را اجرا کنید:
    ./install_yt_downloader.sh

    برای راحتی استفاده، می‌توانید یک دستور مختصر (Alias) در سطح سیستم تنظیم کنید. به عنوان مثال، اگر می‌خواهید از yt به عنوان دستور اصلی استفاده کنید، یکی از این دو کار را انجام دهید:

        ایجاد یک پیوند نمادین در /usr/local/bin/ (نیاز به sudo دارد):
        sudo ln -s /مسیر/کامل/به/install_yt_downloader.sh /usr/local/bin/yt

        یا، یک نام مستعار (alias) به فایل پیکربندی شل خود اضافه کنید (مثلاً ~/.bashrc یا ~/.zshrc):
        alias yt='/مسیر/کامل/به/install_yt_downloader.sh'

    بعد از تنظیم نام مستعار، می‌توانید به سادگی از yt به همراه دستورات فرعی استفاده کنید. به عنوان مثال، برای مشاهده منوی راهنما:
    yt help
